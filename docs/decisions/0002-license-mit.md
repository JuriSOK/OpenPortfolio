# 0002 — Licence open source : MIT

## Statut

Accepté

## Contexte

Le dossier produit prévoit une préparation open source dès le départ (§2.2, §6.1), avec publication générique après stabilisation du portfolio personnel. Une licence doit être choisie dès la Phase 0 pour figurer dans `LICENSE`.

## Décision

La licence retenue est **MIT**, permissive et minimale, adaptée à un projet starter/portfolio destiné à être réutilisé et forké.

Le titulaire du copyright inscrit dans `LICENSE` est **« OpenPortfolio Contributors »** (et non un nom personnel), afin de refléter la nature communautaire visée par le projet dès l'ouverture du dépôt.

## Conséquences

- `LICENSE` (MIT, Copyright (c) 2026 OpenPortfolio Contributors) est créé dès la Phase 0.
- Toute dépendance future doit être compatible avec une redistribution sous licence MIT.
- Cette décision n'engage pas le choix de licence pour un éventuel contenu personnel (textes, médias) publié séparément du moteur générique (cf. principe de séparation moteur / contenu / exemples, dossier produit §7.3, §14.2).
